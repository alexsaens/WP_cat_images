# Extension Icons

This folder should contain the following PNG icons:
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

## Quick Icon Generation

You can create simple icons using any image editor or online tool:

1. **Online Generator**: Use https://favicon.io/ or https://realfavicongenerator.net/
2. **Suggested Design**: A cat paw print with purple/blue gradient (#667eea to #764ba2)
3. **Text Alternative**: "W&P" or a paw emoji

## Temporary Solution

For testing, you can use any 16x16, 48x48, and 128x128 PNG images. The extension will work without icons but will show a default puzzle piece icon.
